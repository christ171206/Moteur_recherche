À propos d'AXORA

Notre mission

AXORA est un moteur de recherche innovant conçu pour fournir des résultats de recherche rapides, précis et pertinents. Notre objectif est de connecter les utilisateurs aux informations dont ils ont besoin de manière efficace et intuitive.

Fonctionnalités

- Recherche web avancée
- Recherche par voix
- Recherche par image
- Suggestions automatiques
- Résultats catégorisés (Web, Images, Vidéos, Actualités)
- Interface utilisateur moderne et responsive

Technologies utilisées

AXORA est construit avec les dernières technologies web : HTML5, CSS3, JavaScript, et un backend Python avec Flask pour une performance optimale.
